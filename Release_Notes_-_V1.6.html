Release Notes - V1.6

V1.6.0 - 6. September 2017

Features

  • Einsatzerfassung / Erstellung eines CIRS Falls mit der Lizenz 'Schnittstelle_EMRIS' SEP-58 EMRIS Schnittstelle

  • Einsatzerfassung / Pauschalfahrten SEP-66 und SEP-67 Pauschalfahrten auswählen

  • Einsatzerfassung / Daten über Sanoweb-Schnittstelle ins Evo importieren und Patientendaten an ELS exportieren SEP-82 SEP-83 Sanoweb Schnittstelle

  • Faktura / Filterverwaltung in der Rechnungsübersicht funktioniert analog der Filterverwaltung in der Einsatzverwaltung SEP-159 / KPSEKSA-15 / KPSESTS-2

Fix

  • Einsatzerfassung / Die 'Ortkategorie' beim Typ 'Spital/Institution', die in den Stammdaten hinterlegt werden kann, werden im Ereignis- und Zielort angezeigt und sind nicht veränderbar SEP-172 / KPSEUST-11

  • Einsatzerfassung / Ein Slider JA/NEIN für die Erfassung der 'Wartezeit vor Ort' ist unter Einsatz/Zeiten eingefügt worden. SEP-212 / KPSEGLR-18 Auswahllisten - Begründungen 'Zeit vor Ort' zu lang

  • Einsatzerfassung / Mit der Eingabe des Sonderzeichens - kann im Filter nach 'leeren Werten' gesucht werden, z.B. zeige alle Einsätze ohne FID SEP-219 / KPSESTS-1 Leer-Einträge suchen

  • Einsatzerfassung / Bei Einsätzen ohne Zielort (Leistungsarten wie 'First Responder' / 'Notarztzubringer' / 'Kein Transport' / etc.) kann die Validierung des Zielortes vom BST-Admin ausgeschaltet werden SEP-224

  • Medizinische Daten / Ist der Button bei 'Messdaten vollständig ermittelt' auf NEIN gestellt, deaktiviert diese Einstellung alle Pflichtfelder in den Medizinischen Daten und verlangt eine Begründung für die Deaktivierung SEP-220

  • Faktura / Auf dem Leistungsbeleg wird unter der Behandlungsart die Einteilung in 'Krankheit' oder 'Unfall' ausgegeben SEP-213 / KPSEGLR-19

  • Faktura / STS - 'Reduktion Mehrfachtransport' wird nicht korrekt abgerechnet, wenn Patient erst gelöscht und dann nochmals erfasst worden ist SEP-228 / KPSESTS-40

Enhancements

  • Einsatzverwaltung / Wie sind die Folgeeinsätze miteinander verknüpft SEP-165 / KPSEUST-10 Hat Folgeeinsatz / Ist Folgeeinsatz

  • Medizinische Daten / Die Auswahl für 'Autodrehleiter ADL' in 'Bergung / Lagerung' ergänzt SEP-167 / KPSEUST-21

  • Medizinische Daten / Diverse Rechtschreibfehler sind korrigiert worden SEP-168 SEP-169

  • Medizinische Daten / Bei der Diagnose 'Commotio cerebri' ist das Detail 'Körperregion' entfernt worden SEP-174 / KPSERSG-61

  • Medizinische Daten / ComboBox EKG, Erweiterung 'Schenkelblock links / rechts' SEP-216 / KPSEUST-3

  • Medizinische Daten / ComboBox Auskultation, neue Einträge 'Abgeschwächtes Atemgeräusch links/rechts' SEP-217 / KPSEUST-34

  • Auswertung / Die Excel-Vorlagen sind in der Tabelle korrekt beschriftet SEP-158

  • Dashboard / Zahlen und Fakten auf dem Dashboard verwalten
