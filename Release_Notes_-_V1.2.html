Release Notes - V1.2

V1.2.0 - 01. November 2016

Features

  • Die Rechnungsübersicht ist nun verfügbar. (3778)

  • Tarifkataloge und Leistungen können verwaltet werden. (4006)

  • Es können benutzerspezifische Filter angelegt werden. Globale Filter wurden entfernt. (4389)

  • Die PatPool(HL7)-Schnittstelle wurde implementiert und die Benutzeroberfläche entsprechend angepasst. (4657, 4658)

  • Die Sichtbarkeit von Ereignissen kann nun zugunsten des Datenschutzes eingeschränkt werden. (14920)

  • Die Leistungsarten können neu in den Stammdaten verwaltet werden. (14991)

  • Einer Ortschaft kann neu eine Organisation, eine Spitalregion und ein zuständiges Zielspital zugewiesen werden. (15337)

  • Spitäler können einer Spitalregion zugeordnet werden. (15339)

  • Die Leistungsdatenexportschnittstellen ALIS und SD wurden implementiert.

  • Für die Installation der Clients steht per sofort ein MSI (Setup) zur Verfügung.

Fix

  • Diverse Verbesserungen im Bereich der Einsatzerfassung.

Enhancements

  • Die erfassten Zeiten auf der Erfassungsmaske werden nun validiert. (4568)

  • Im Filter kann nun auch nach Wertebereichen gefiltert werden. (4682)

  • Im Standard wird dem Benutzer nun die gespeicherten Filter 'aktuelles Quartal' und 'letztes Quartal' angezeigt. (4821)

  • Institutionen können neu als Zielorte definiert werden, damit diese unter 'Zielort' in der Erfassungsmaske angezeigt werden können. (15246)

  • Diverse Verbesserungen im Bereich der 'Medizinischen Daten'. (15370, 15371, 15372, 15375, 15384, 15388, 15392)
