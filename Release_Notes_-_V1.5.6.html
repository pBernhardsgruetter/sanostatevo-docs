Release Notes - V1.5.6

V1.5.6 - 14. Juli 2017

Fix

  • Einsatzerfassung / Filterung nach der Einsatzart 'Primär/Sekundär' ist nun immer möglich. SEP-130 / KPSERSG-52

  • Einsatzerfassung / Nach dem Starten der Software steht der Cursor nun standardmässig im Feld 'Benutzername' SEP-131 / KPSESTS-5

  • Einsatzverwaltung / Bei Ereignissen mit mehreren Einsatzmitteln, erscheinen diese nun korrekt auf dem Leistungsbeleg. SEP-144 / KPSESTS-30

  • Einsatzverwaltung / Bei der Berechnung des Nachtzuschlags wird das 'nicht verrechnen Flag' der Einsatzfunktion berücksichtigt. SEP-147 / KPSESTS-32

  • Auswertung / Die Benennung der einzelnen Excel-Sheets (Tabelle 1) wurde korrigiert. SEP-158

  • RSG: ALIS-Export / Die Konfiguration der Mail-Einstellungen der ALIS-Schnittstelle werden korrekt übernommen. SEP-146 / KPSERSG-91

  • STS: Auswertung / Bei der Einsatzliste mit Grunddaten und Diagnose wurde die Spalte 'Dringlichkeit' hinzugefügt. SEP-152 / KPSESTS-35

Enhancements

  • RSG: Einsatzerfassung / Bei Einsätzen mit S1 wurde die Hilfsfristenvalidierung deaktiviert. SEP-154 / KPSERSG-94

  • STS: Einsatzverwaltung / Der Abfahrtsort ist auf dem Leistungsbeleg ersichtlich. SEP-148 / KPSESTS-34

  • Medizinische Daten / Es ist nun möglich einen Blutzucker-Wert bis maximal 50 mmol/l zu erfassen. SEP-150 / KPSESTS-22

  • KSGL: Medizinische Daten / Pflichtfelder und GUI-Regeln wurden angepasst SEP-117 / KPSEGL-10

  • STS: Medizinische Daten / Beim Karteireiter 'Psyche' wird das Feld 'Mit FU' standardmässig auf [Nein] gesetzt. SEP-135 / KPSESTS-21

  • PatPool-Schnittstelle / PatPool-Actions, welche ignoriert werden sollen, können in den Einstellungen konfiguriert werden. SEP-95

  • HL7-Importer / Die Action A03 (Austritt) wird nun korrekt abgehandelt. SEP-141 / KPSERSG-11

  • HL7-Importer / Die Entkoppelung PID- und PV1-Segmente (Pat/Fall) für die Action A08 (Update) wurde implementiert. SEP-142 / KPSERSG-11
